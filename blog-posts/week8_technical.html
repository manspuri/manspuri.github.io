Recursion

Recursion is the process of a function calling itself within its body. Below is an example of recursion.

Recursive functions are most commonly used in the form of a loop. In this case, we print 0, and then put 0+1, or 1, back into the count() function. If 1 is put into the function, that means 1+1 is then put back into the count() function. And so on... This is an infinite loop because there is nothing within the function that stops it from calling itself over and over again - there is no point at which the function is told to STOP!

For recursion to properly work, there must be some escape route for this function. To do this, the argument (either the first value put into the function or the return value) needs to pass a test of some sort. To make our infinite loop finite, we can set a maximum return value. This loop-hole or escape is sometimes called the base case or the point where the function does not call itself again.
