Classes Bollywood Style

Ruby is an object-oriented programming language which has Classes and Objects. Tutorialspoint defines classes as "the blueprint from which individual objects are created." Some of the objects which a class can have are instance variables and instance methods. Instance variables are like the characteristics of the class and instance methods are functions of that class.

You might be wondering why we have classes in the first place. Classes are useful when you have a large amount of information that can be organized. For example, what if you have a huge collection of Bollywood films and you'd like to compile their titles, dates of release, and starring actors? You could create a variable for each title of each movie OR you could create an instance variable of '@title' for all Bollywood movie titles. Classes allow you to apply characteristics and functions on a larger scale.
